# 🧪 Lab 3 – Test Design Techniques

## 🎯 Objective

The purpose of this laboratory is to apply different **Test Design Techniques** to create effective and optimized test cases. These techniques help reduce the number of test cases while maintaining adequate test coverage.

---

## 📚 Techniques Covered

| Technique | Description |
|-----------|-------------|
| **Equivalence Partitioning (EP)** | Divides input data into valid and invalid partitions to reduce the number of test cases. |
| **Boundary Value Analysis (BVA)** | Tests values at the boundaries of input ranges where defects are most likely to occur. |
| **Decision Table Testing (DT)** | Verifies application behavior for different combinations of input conditions and business rules. |
